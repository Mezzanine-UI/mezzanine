# DateRangePicker

<Description of={DateRangePicker} />

### For typescript users

When using date-range-picker, you should declare an exported `DateType` with your desired date library.
For example, having below code in your `index.d.ts`:

```ts
import { Moment } from 'moment';

declare module '@mezzanine-ui/core/calendar' {
  export type DateType = Moment;
}
```
