## Overview

CardGroup is a container component for laying out multiple cards in a horizontal row with consistent spacing. It uses CSS Grid for layout management.

## Usage

CardGroup validates its children and only accepts card components (like BaseCard). Invalid children will be filtered out with a console warning in development.
